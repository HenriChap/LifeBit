# WDL Dev Helper
## A didactic code for WDL developers 
### WDL
This repository has a didactic code with instructions, concepts, and tips for WDL development and best practices, based on [WDL docs](https://docs.openwdl.org/en/latest/).
For easier navigation, use go to the de_helper.wdl file and use the search tool (Ctrl+f or Cmd+f). Exclamation marks (!) in the commentaries are for tips and section sign (ยง) are content explanations.
